# Contributing to XPanel

Thank you for your interest in contributing to **XPanel** 🚀

We welcome contributions from the community to help improve the project.

---

## 🧩 Ways to Contribute

- Report bugs
- Suggest new features
- Improve documentation
- Submit Pull Requests
- Review code

---

## 🛠 Development Stack

- Panel: Laravel (PHP)
- Daemon: Go
- Services: Docker / Docker Compose
- CLI: Bash

---

## 📦 Project Structure

- `/panel` → Laravel web panel
- `/daemon` → Go background services
- `/installer` → Installer & CLI
- `/docs` → Documentation

---

## 🔄 Workflow

1. Fork the repository
2. Create a feature branch  
   ```bash
   git checkout -b feature/my-feature
    ```
3. Commit changes with clear messages
4. Push to your fork
5. Open a Pull Request

## 📐 Code Style

- Keep code clean and readable
- Follow existing patterns
- Avoid breaking backward compatibility

## 🤝 Community Rules

- Be respectful
- Be constructive
- Help others

Thank you for helping build XPanel ❤️
